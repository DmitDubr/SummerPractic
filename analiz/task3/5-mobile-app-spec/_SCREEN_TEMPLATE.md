## Используемые запросы

### {operationId}

**Метод:** {GET | POST | PATCH | …}  
**Путь:** `{path}`  
**Спецификация:** [../api/openapi.yaml](../api/openapi.yaml) → `{operationId}`

**Обработка ответа:**

| HTTP / код | UI-реакция |
| :-- | :-- |
| 200 / 201 + data | Content / переход |
| 200 + empty | Empty state («Пока нет доступных заездов» — FR-005) |
| 409 `{ErrorCode}` | SCR-007 или inline (см. [api-sequence.md](../4-design/api-sequence.md)) |
| 4xx / 5xx / timeout | Error + «Повторить» |

**Доменные коды createBooking:** `NO_SPOTS`, `SLOT_CANCELLED`, `RENTAL_UNAVAILABLE`, `SLOT_REBOOK_FORBIDDEN`.

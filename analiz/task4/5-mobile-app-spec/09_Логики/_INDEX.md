## Отличия от «Апекс» (task3)

| Аспект | «Глина» (task4) | «Апекс» (task3) |
| :-- | :-- | :-- |
| Платформа | Android (NFR-001) | iOS |
| Правило отмены | ≥ **3 ч** (FR-014) | ≥ **1 ч** |
| Фильтры | Время, уровень, **программа** | Время, **маршал** |
| Рейтинги в UI | **MVP** (FR-023) | v2 |
| Push | **MVP** (FR-024) | v2 |
| Экипировка | Отдельная логика L-009; прокат исчерпан → «со своим» | Прокат шлема; исчерпание → блок |
| Участники в брони | **1** (FR-010) | Stepper N картов |
| Статус форс-мажора | «Отменено мастерской» | «Отменён центром» |

## Обзор

Экран показывает полную информацию о выбранном мастер-классе: время, длительность (2–2,5 ч), программу, мастера, рейтинг, доступность мест и цену. Это точка входа в запись из расписания.

### User Story

> Как клиент, я хочу увидеть детали занятия, чтобы понять, подходит ли мне мастер-класс, и перейти к записи.

**Не в MVP:** лист ожидания при `NO_SPOTS` (FR-011) — только возврат к расписанию; фильтр по мастеру; номер гончарного круга; iOS.

## Используемые запросы

### getSlot

**Метод:** GET  
**Путь:** `/slots/{slotId}`  
**Спецификация:** [../../api/openapi.yaml](../../api/openapi.yaml) → `getSlot`

**Обработка ответа:**

| HTTP / код | UI-реакция |
| :-- | :-- |
| 200 + data (`freeSpots > 0`) | Content + бейдж «Есть места» + активный CTA «Записаться» |
| 200 + data (`freeSpots = 0`) | Content + бейдж «Мест нет», CTA скрыт |
| 200 + data (`rentalAvailability.rentalFullyExhausted = true`) | Инфо «Прокат закончился, можно записаться со своим снаряжением»; CTA остаётся активным при `freeSpots > 0` |
| 404 | Error state + «К расписанию» |
| 5xx / timeout | Error state + «Повторить» |

## Элементы экрана

| Элемент | Описание | Источник данных | Валидация / поведение |
| :-- | :-- | :-- | :-- |
| Кнопка «Назад» | Возврат к расписанию | Навигация | Возврат без потери контекста списка |
| Дата и время | Старт занятия | `slot.startsAt` | Формат «Сб, 5 июля · 11:00» |
| Длительность | Продолжительность занятия | `slot.durationMinutes` | Отображать как `~2 ч` или `~2,5 ч` (диапазон 120–150 мин) |
| Бейдж программы | Тип программы | `slot.program.typeName` | «Лепка» / «Работа на круге» |
| Уровень | Уровень занятия | `slot.level` | «Начинающий» / «Средний» / «Продвинутый» |
| Бейдж доступности | Наличие мест | `slot.freeSpots` | «Есть места» при `freeSpots > 0`; «Мест нет» при `freeSpots = 0`. Счётчик X/Y и индикатор заполненности **не показываются** |
| Карточка мастера | Имя и рейтинг | `slot.master.*` | Если `ratingCount = 0` → «Пока нет оценок» |
| Описание программы | Краткое содержание занятия | `slot.programInfo.summary` | 1–3 строки, остальное обрезать |
| Блок цены | Стоимость занятия | `slot.programInfo.price` / `slot.price` | Цена зависит только от программы |
| Инфо по прокату | Доступность проката | `slot.rentalAvailability` | При полном исчерпании — информационный текст про запись «со своим»; **не** статус «Недоступно» |
| CTA «Записаться» | Переход к форме | Локально от `slot` | Доступен только если `freeSpots > 0` и `status = OPEN` |
| Бейдж «Мест нет» | Слот заполнен | Локально от `slot` | Показывается при `freeSpots = 0`; waitlist CTA отсутствует |

**Терминология:** **мастер**, **программа**, **мастер-класс**, **занятие**, **мастерская**; прокат (инструменты, фартук) **не влияет на цену** (FR-012). Термины «шеф», «тип кухни», «аллергии» не использовать.

## Состояния экрана

| Состояние | Условие | Отображение |
| :-- | :-- | :-- |
| Loading | `getSlot` в процессе | Skeleton блоков + placeholder CTA |
| Content | 200 + data | Полный контент экрана |

## Критерии приёмки

| ID | Критерий |
| :-- | :-- |
| AC-001 | **Дано** пользователь открыл занятие из SCR-001, **Когда** `getSlot` вернул 200, **Тогда** показываются дата, программа, мастер, бейдж доступности и цена. |
| AC-002 | **Дано** `freeSpots > 0` и `status = OPEN`, **Когда** экран в Content, **Тогда** CTA «Записаться» активен и ведёт в SCR-005 с `slotId`. |
| AC-003 | **Дано** `freeSpots = 0`, **Когда** экран загружен, **Тогда** показывается бейдж «Мест нет», CTA записи и CTA waitlist отсутствуют; счётчик X/Y не отображается. |
| AC-004 | **Дано** `rentalAvailability.rentalFullyExhausted = true` и `freeSpots > 0`, **Когда** экран загружен, **Тогда** показан информационный текст про запись «со своим», CTA «Записаться» остаётся активным. |
| AC-005 | **Дано** `getSlot` вернул 404, **Когда** открыт экран, **Тогда** отображается Error с возможностью перейти к расписанию. |
| AC-006 | **Дано** нет сети, **Когда** открывается экран, **Тогда** показывается офлайн-ошибка и кнопка «Повторить». |
| AC-007 | **Дано** у мастера `ratingCount = 0`, **Когда** отображается карточка мастера, **Тогда** показывается текст «Пока нет оценок». |
| AC-008 | **Дано** `durationMinutes` в диапазоне 120–150, **Когда** экран отображает длительность, **Тогда** показано `~2 ч` или `~2,5 ч`, а не `~3 ч`. |
| AC-009 | **Дано** `freeSpots > 0`, **Когда** экран отображает доступность, **Тогда** показан бейдж «Есть места» без счётчика мест и номера круга. |

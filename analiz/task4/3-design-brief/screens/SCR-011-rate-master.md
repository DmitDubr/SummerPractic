## Обзор

Bottom sheet для оценки мастера после посещённого занятия. Экран поддерживает только выбор звёзд 1-5 и отправку `createOrUpdateMasterRating`; текстовый отзыв, теги и другие поля в MVP не используются.

Оценка доступна только при `Booking.status = ATTENDED` и только в пределах 7 дней после окончания занятия. Модель оценки — upsert по паре клиент-мастер: повторная отправка обновляет существующую оценку (FR-022).

### User Story

> Как клиент, я хочу быстро поставить оценку мастеру после занятия, чтобы повлиять на его публичный рейтинг без длинного отзыва.

**Не в MVP:** текстовые отзывы; отдельный полноэкранный экран оценки; оценка до посещения; оценка после 7 дней.

## Используемые запросы

### createOrUpdateMasterRating

**Метод:** POST  
**Путь:** `/ratings`  
**Спецификация:** [../../api/openapi.yaml](../../api/openapi.yaml) → `createOrUpdateMasterRating`

**Обработка ответа:**

| HTTP / код | UI-реакция |
| :-- | :-- |
| 201 + data | Новая оценка создана, sheet закрывается, toast «Спасибо за оценку!» |
| 200 + data | Оценка обновлена (upsert), sheet закрывается, на SCR-009 показывается обновлённое значение |
| 400 `VALIDATION_ERROR` | Inline ошибка, кнопка снова активна |
| 403 `BOOKING_NOT_ATTENDED` | Sheet закрывается, на SCR-009 сообщение «Оценить можно после посещения» |
| 403 `RATING_PERIOD_EXPIRED` | Sheet закрывается, на SCR-009 сообщение «Срок оценки истёк» |
| 401 / 5xx / timeout | Error состояние с retry, выбранные звёзды сохраняются локально |

## Элементы экрана

| Элемент | Описание | Источник данных | Валидация / поведение |
| :-- | :-- | :-- | :-- |
| Drag handle | Индикатор bottom sheet | UI | Закрытие свайпом вниз |
| Аватар мастера | Круглое фото/placeholder | `masterAvatarUrl` | Placeholder при `null` |
| Имя мастера | Заголовок карточки | `masterName` | Обязательное поле, без сокращений «инструктор» |
| Блок звёзд | 5 tappable иконок | Локальный state | Допустимы только целые 1-5 |
| CTA «Отправить» | Основное действие | Локальный state | Disabled при `stars=null`; в Submitting блокирует повторный тап |
| Inline сообщение | Текст ошибки/успеха | API response | Успех: краткое «Спасибо за оценку!» |

**Терминология:** **мастер**, **программа занятия** (лепка / работа на круге), **мастерская «Глина»**; прокат (инструменты, фартук) **не влияет на цену** (FR-012).

## Состояния экрана

| Состояние | Условие | Отображение |
| :-- | :-- | :-- |
| Loading | Краткая инициализация sheet | Контент без блокирующего спиннера |

## Критерии приёмки

| ID | Критерий |
| :-- | :-- |
| AC-001 | **Дано** бронь со статусом `ATTENDED` и с момента занятия прошло не более 7 дней, **Когда** клиент открывает SCR-011, **Тогда** экран доступен и позволяет выбрать 1-5 звёзд. |
| AC-002 | **Дано** клиент не выбрал ни одной звезды, **Когда** открыт SCR-011, **Тогда** кнопка «Отправить» неактивна. |
| AC-003 | **Дано** клиент отправляет оценку с валидным `bookingId`/`masterId`, **Когда** API возвращает 201, **Тогда** sheet закрывается и на SCR-009 отображается сохранённая оценка. |
| AC-004 | **Дано** ранее уже была оценка этого мастера текущим клиентом, **Когда** выполняется повторная отправка в окне 7 дней, **Тогда** вызывается `createOrUpdateMasterRating` и при 200 обновляется существующая оценка (upsert, FR-022). |
| AC-005 | **Дано** прошло более 7 дней после занятия, **Когда** клиент пытается открыть SCR-011, **Тогда** оценка недоступна и показывается сообщение «Оценить можно в течение недели после занятия». |
| AC-006 | **Дано** статус брони не `ATTENDED`, **Когда** клиент открывает экран оценки по CTA или deep link, **Тогда** отправка блокируется и показывается сообщение «Оценить можно после посещения занятия». |
| AC-007 | **Дано** пользовательский интерфейс SCR-011, **Когда** пользователь взаимодействует с формой, **Тогда** доступны только звёзды 1-5 без поля текстового отзыва, тегов и иных дополнительных полей. |
| AC-008 | **Дано** экран в состоянии Content, **Когда** пользователь тапает по звёздам, **Тогда** каждая звезда имеет tappable-зону не менее 44dp и визуально различаются выбранное/невыбранное состояние. |

## Обзор

Bottom sheet фильтрации расписания на SCR-001 по трём критериям MVP: **программа** (лепка / работа на круге), **время суток**, **уровень**. Фильтр **по мастеру не реализуется**. При «Применить» sheet закрывается и передаёт параметры в SCR-001 для `listSlots`.

### User Story

> Как клиент, я хочу отфильтровать занятия по программе, времени и уровню, чтобы быстрее найти подходящий мастер-класс.

**Не в MVP:** фильтр по мастеру, iOS.

## Используемые запросы

### listPrograms

**Метод:** GET  
**Путь:** `/programs`  
**Спецификация:** [../../api/openapi.yaml](../../api/openapi.yaml) → `listPrograms`

**Обработка ответа:**

| HTTP | UI-реакция |
| :-- | :-- |
| 200 + items | Чекбоксы/чипы программ (`isActive = true`): «Лепка», «Работа на круге» |
| 500 / timeout | Inline «Не удалось загрузить программы» + «Повторить» в sheet |
| 200 + empty | Секция программ скрыта или текст «Справочник недоступен» |

### listSlots (косвенно через SCR-001)

**Query после «Применить»:**

| Параметр | Источник |
| :-- | :-- |
| `programIds` | Выбранные программы (OR внутри группы) |
| `timeOfDay` | Сегмент времени суток |
| `level` | Выбранный уровень |

Период (`dateFrom`/`dateTo`) **не сбрасывается** при смене фильтров занятий (LOGIC-005).

## Элементы экрана

| Элемент | Описание | Источник данных | Валидация / поведение |
| :-- | :-- | :-- | :-- |
| Drag handle | Свайп закрытия | UI | Без применения |
| Секция «Программа» | Multi-select чипы/чекбоксы | `listPrograms.items[]` | OR внутри группы; значения: лепка, работа на круге |
| Секция «Время суток» | Single-select | Локально | `morning` 06–11:59, `afternoon` 12–16:59, `evening` 17–23:59 |
| Секция «Уровень» | Single-select | Локально | `BEGINNER` / `INTERMEDIATE` / `ADVANCED` |
| «Сбросить» | Сброс всех фильтров sheet | Локально | Не сбрасывает период SCR-002 |
| «Применить» | Подтверждение | Локально | Закрывает sheet → reload SCR-001 |

**Badge на SCR-001:** число заполненных категорий (0–3), не число выбранных программ.

## Критерии приёмки

| ID | Критерий |
| :-- | :-- |
| AC-001 | **Дано** открыт SCR-003, **Когда** sheet открывается, **Тогда** вызывается `listPrograms` и отображаются активные программы (лепка, работа на круге). |
| AC-002 | **Дано** выбраны «Лепка» и «Вечер», **Когда** «Применить», **Тогда** SCR-001 перезагружает `listSlots` с `programIds` и `timeOfDay=evening`, badge = 2. |
| AC-003 | **Дано** выбран только уровень «Начинающий», **Когда** «Применить», **Тогда** query содержит `level=BEGINNER`, badge = 1. |
| AC-004 | **Дано** активны фильтры, **Когда** «Сбросить», **Тогда** query без `programIds`/`timeOfDay`/`level`, badge = 0. |
| AC-005 | **Дано** `listSlots` вернул пусто при фильтрах, **Когда** SCR-001, **Тогда** «Ничего не найдено» + «Сбросить фильтры». |
| AC-006 | **Дано** изменён период на SCR-002, **Когда** применены фильтры SCR-003, **Тогда** оба параметра переданы в один `listSlots`. |
| AC-007 | **Дано** ошибка загрузки программ, **Когда** retry в sheet, **Тогда** повторный `listPrograms` без закрытия sheet. |

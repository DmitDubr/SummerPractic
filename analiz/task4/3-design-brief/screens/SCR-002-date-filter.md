## Обзор

Bottom sheet для выбора периода показа расписания занятий мастерской «Глина». Сам sheet не делает собственных API-запросов: сохраняет выбранный диапазон и возвращает `dateFrom/dateTo` в SCR-001, где выполняется `listSlots`.

### User Story

> Как клиент, я хочу выбрать произвольный диапазон дат, чтобы посмотреть мастер-классы не только на ближайшие 7 дней.

**Не в MVP:** пресеты периодов, автозагрузка без нажатия «Применить», iOS.

## Используемые запросы

### listSlots (косвенный вызов из SCR-001 после применения периода)

**Метод:** GET  
**Путь:** `/slots`  
**Спецификация:** [../../api/openapi.yaml](../../api/openapi.yaml) → `listSlots`

**Query-параметры, которые передаёт SCR-002:**

| Параметр | Значение |
| :-- | :-- |
| `dateFrom` | Выбранная дата начала |
| `dateTo` | Выбранная дата окончания |

**Обработка ответа (на стороне SCR-001):**

| HTTP / код | UI-реакция |
| :-- | :-- |
| 200 + data | SCR-001 показывает список занятий в выбранном диапазоне |
| 200 + empty | SCR-001 показывает Empty («Пока нет доступных занятий» / «Ничего не найдено») |
| 400 | SCR-001 показывает Error по параметрам периода |
| 500 / timeout | SCR-001 показывает Error + «Повторить» |

## Критерии приёмки

| ID | Критерий |
| :-- | :-- |
| AC-001 | **Дано** открыт SCR-002 из SCR-001 с дефолтным периодом, **Когда** sheet отображается, **Тогда** предустановлен диапазон «сегодня + 6 дней». |
| AC-002 | **Дано** пользователь выбрал только дату начала, **Когда** дата окончания не выбрана, **Тогда** кнопка «Применить» неактивна. |
| AC-003 | **Дано** пользователь выбрал валидный диапазон, **Когда** нажал «Применить», **Тогда** sheet закрывается и в SCR-001 передаются `dateFrom/dateTo`. |
| AC-004 | **Дано** пользователь нажал «Сбросить», **Когда** действие подтверждено, **Тогда** в SCR-001 применяется дефолтный режим 7 дней без `dateFrom/dateTo`. |
| AC-005 | **Дано** пользователь закрыл sheet свайпом вниз, **Когда** возврат в SCR-001 завершён, **Тогда** период остаётся последним применённым (без изменений). |
| AC-006 | **Дано** после применения диапазона `listSlots` вернул пусто, **Когда** открыт SCR-001, **Тогда** отображается корректный Empty state «Пока нет доступных занятий». |
| AC-007 | **Дано** нет сети при нажатии «Применить», **Когда** SCR-001 пытается загрузить расписание, **Тогда** показывается ошибка загрузки и доступен retry. |

# FEAT-001 — Маска ввода телефона

## Цель

Единый формат номера при записи и в листе ожидания: `+7 (900) 123-45-67`.

## Поведение

- При вводе цифры автоматически расставляются скобки и дефисы.
- Допускается вставка «сырого» номера (`89001234567`, `+79001234567`) — приводится к маске.
- На API уходит нормализованный `+79001234567` (`PhoneValidator.normalize`).
- Клавиатура: `Phone`.
- Подсказка под полем: «Формат: +7 (900) 123-45-67».

## Реализация

| Компонент | Путь |
|-----------|------|
| `PhoneMaskFormatter` | `domain/policy/PhoneMaskFormatter.kt` |
| `PhoneTextField` | `presentation/components/PhoneTextField.kt` |
| Экраны | `BookingFormScreen`, `WaitlistScreen` |

## Критерии приёмки

- [x] Маска при посимвольном вводе
- [x] Корректная нормализация при submit
- [x] Профиль с `+7900…` отображается в маске при открытии формы

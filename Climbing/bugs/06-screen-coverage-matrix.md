# 06 — Матрица покрытия SCR-001–013

| ID | Экран | Статус | Бэкенд API | Клиент UI | Пробелы |
|----|-------|:------:|:----------:|:---------:|---------|
| SCR-001 | Расписание | ⚠️ | ✅ | ✅ | CMP-13 (PTR), empty CTA |
| SCR-002 | Фильтр дат | ⚠️ | ✅ | ⚠️ | CMP-12 (календарь) |
| SCR-003 | Фильтры слотов | ⚠️ | ✅ | ⚠️ | CMP-11 (★ на чипах) |
| SCR-004 | Деталь слота | ⚠️ | ✅ | ✅ | CMP-11 (рейтинг инструктора) |
| SCR-005 | Оформление записи | ✅ | ✅ | ✅ | CMP-02 (inline вместо sheet) |
| SCR-006 | Успешная запись | ⚠️ | ✅ | ⚠️ | CMP-03, CMP-04 (dialog, no push) |
| SCR-007 | Ошибка записи | ✅ | ✅ | ✅ | TZ-INT-01 (имя кода в ТЗ) |
| SCR-008 | Мои записи | ⚠️ | ✅ | ⚠️ | CMP-05, CMP-08, CMP-13 |
| SCR-009 | Деталь записи | ⚠️ | ✅ | ⚠️ | CMP-06, CMP-07, CMP-09, CMP-10 |
| SCR-010 | Подтверждение отмены | ⚠️ | ✅ | ⚠️ | CMP-10, CMP-14 |
| SCR-011 | Оценка инструктора | ❌ | ✅ | ❌ | CMP-01 |
| SCR-012 | Лист ожидания | ⚠️ | ✅ | ⚠️ | CMP-06, CMP-07, CMP-18 |
| SCR-013 | Контактные данные | ❌ | ✅ | ❌ | CMP-02 |

**Легенда:** ✅ соответствует · ⚠️ частично · ❌ не реализовано

---

## Детали по экранам с пробелами

### SCR-001 — Расписание

| Элемент ТЗ | Статус |
|------------|:------:|
| Группировка по дням, sticky headers | ✅ |
| Чипы периода и фильтров | ✅ |
| Карточка: время, формат, инструктор, ★, места, цена | ⚠️ |
| Pull-to-refresh | ❌ |
| Empty «Пока нет доступных тренировок» | ✅ |
| Skeleton loading | ✅ |

### SCR-008 — Мои записи

| Элемент ТЗ | Статус |
|------------|:------:|
| Сегмент «Предстоящие \| Прошедшие» | ❌ |
| Бейджи всех статусов | ⚠️ только Active/Waitlist |
| Офлайн-баннер + кэш | ❌ |
| Pull-to-refresh | ❌ |
| Empty + CTA «Посмотреть расписание» | ⚠️ |

### SCR-009 — Деталь записи

| Элемент ТЗ | Статус |
|------------|:------:|
| Блок информации о тренировке | ⚠️ |
| Блок «Ваша запись» (снаряжение) | ❌ |
| Причина отмены скалодромом | ❌ |
| CTA по статусу (отмена / перезапись / оценка / leave waitlist) | ⚠️ только отмена Active |
| Deep link из push | ❌ |
| Офлайн просмотр кэша | ❌ |

### SCR-012 — Лист ожидания

| Элемент ТЗ | Статус |
|------------|:------:|
| Вступление в очередь | ✅ |
| Экран подтверждения «Вы в очереди» | ⚠️ упрощён |
| Позиция в очереди | ❌ |
| Покинуть очередь | ❌ |
| Push → SCR-005 | ❌ |
